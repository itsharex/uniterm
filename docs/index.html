uniTerm

一款现代化跨平台终端模拟器，内置 AI 助理功能。 A modern cross-platform terminal emulator with built-in AI assistant.

下载安装 Download GitHub

功能特性Features

SSH 客户端Client

密码/私钥认证、多标签页管理、5 种配色、可定制字体与操作Password & key auth, multi-tab sessions, 5 color schemes, customizable fonts & behavior

SFTP 文件管理File Manager

双栏浏览本地与远程文件，上传下载拖拽，传输任务按标签跟踪Dual-pane browse, upload, download, drag-and-drop, per-tab transfer tracking

AI 助理Assistant

侧边对话、终端执行命令，三种确认模式，多轮对话持久化Sidebar chat, terminal command execution, 3 confirm modes, persistent sessions

工作区分屏Workspace & Split

标签合并为工作区，水平/垂直分屏，拖拽自由调整布局Merge tabs into workspaces, horizontal/vertical splits, drag to rearrange

连接管理Connection Manager

分组、搜索、批量操作，多选/范围选择连接Grouping, search, batch ops, multi-select & range-select connections

主题国际化Themes & i18n

暗色/深蓝/浅色主题，跟随系统切换，中英双语Dark, deep blue, light themes with system auto-detect, bilingual UI
